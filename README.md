# rayx-component-pro-vue 
快速搭建你的组件库

这是一套基于vue的组件库脚手架, 脚手架的工作是对组件和md文件进行解析，生成文档界面，本身并不包含组件。

换句话说，按脚手架规范开发组件，即可快速搭建一套带文档界面的组件库。

组件开发规范请在启动dev模式后查看

## 如何使用

安装依赖
```
yarn 或者 npm i
```

开启dev
```
yarn dev 或者 npm run dev
```

打包库
```
yarn lib 或者 npm run lib
```
