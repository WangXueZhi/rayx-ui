# component-pro-vue 
基于vue, 开箱即用的组件库脚手架

## 如何使用

安装依赖
```
yarn 或者 npm i
```

开启dev
```
yarn dev 或者 npm run dev
```

打包库
```
yarn lib 或者 npm run lib
```
